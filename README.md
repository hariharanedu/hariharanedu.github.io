## How to Use

### Local Development

1. Simply open `index.html` in your web browser
2. No build process or dependencies required!

### Deploy to GitHub Pages

1. Create a new repository on GitHub
2. Upload all files (index.html, styles.css, script.js)
3. Go to repository Settings > Pages
4. Select "Deploy from a branch"
5. Choose "main" branch and "/ (root)" folder
6. Click Save
7. Your site will be live at: `https://yourusername.github.io/repository-name/`

## Customization

### Change Colors
Edit the gradient colors in `styles.css`:
- Primary gradient: `#8a2be2, #4169e1, #00bfff`
- Adjust planet colors in `.planet-1` through `.planet-4` classes

### Adjust Animations
Modify animation speeds in `styles.css`:
- Stars: `move-twink-back` animation duration
- Moon rotation: `rotate3d` animation duration
- Black hole spin: `spin` animation duration

### Add More Content
Edit `index.html` to add more sections, planets, or text content.
